Розмітка заголовків

Заголовки впорядковують текст, форматуючи його візуальні структуру. При вікористанні заголовків необхідно врахувати їх ієрархію. В загловки не можна вкладати інші елементи
<h1> - заголовок найвищого рівня, на сторінці рекомендовано використовувати не більше одного разу.
<h2>, <h3>, <h4>, <h5>, <h6> - заголовки нижчик рівнів, розташовані за спадання їх важливості.

Синтаксис:

<h1>Заголовок 1-го рівня </h1>
<h2>Заголовок 2-го рівня </h2>
<h3>Заголовок 3-го рівня </h3>
<h4>Заголовок 4-го рівня </h4>
<h5>Заголовок 5-го рівня </h5>
<h6>Заголовок 6-го рівня </h6>

Результат:

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня




Теги для форматування тексту

<strong> - вказує на важливість фрагменту тексту. Зазвичай виділяє текст напівжирним.

Синтаксис:

Тег визначає <strong>ВАЖЛИВІСТЬ</strong> визначеного тексту

Результат:

Тег визначає важливість визначеного тексту

<em> - вказує на логічне посилання фрагменту тексту, уточнює зміст виразу. Зазвичай виділяє текст курсивом.

Синтаксис:

HTML - <em>ГОЛОВНИЙ</em> інструмент веб-розробника

Результат:

HTML - головний інструмент веб-розробника

<small> - зменшує розмір шрифту на одиницю порівняно з простим текстом.

Синтаксис:

З сімейства котячих найбільша кішка це зовсім не <small>ЛЕВ</small>, а <small>ТИГР УССУРІЙСЬКИЙ</small>.

Результат:

З сімейства котячих найбільша кішка це зовсім не лев, а тигр уссурійський.

<ins> - призначений для виділення тексту, який було додано до нової версії документа.

Синтаксис:

У наступних версіях <ins>ПІДХІД РОЗРОБНИКІВ ЦЬОГО БРАУЗЕРА БУЛО ЗМІНЕНО</ins>.

Результат:

У наступних версіях підхід розробників цього браузера було змінено.

<del> - використовується для виділення тексту, видаленого в новій версії документа.

Синтаксис:

7 січня 2021 року, засновник компаній Tesla та SpaceX <del>ДЖЕФФ БЕЗОС;/del> Ілон Маск став найбагатшою людиною у світі.

Результат:

7 січня 2021 року, засновник компаній Tesla та SpaceX Джефф Безос Ілон Маск став найбагатшою людиною у світі.

<sub> - відображає шрифт у вигляді нижнього індексу. Текст при цьому розташовується нижче за базову лінію інших символів рядка і зменшеного розміру.

Синтаксис:

Під дією гідроксиламіну альдегіди перетворюються на оксими: СН<sub>3</sub>СНО + NH<sub>2</sub>OH = CH<sub>3</sub>C(=NOH)H + H<sub>2</sub>O

Результат:

Під дією гідроксиламіну альдегіди перетворюються на оксими: СН3СНО + NH2OH = CH3C(=NOH)H + H2>O

<sup> - відображає шрифт у вигляді верхнього індексу. Шрифт при цьому відображається вище базової лінії тексту та зменшеного розміру.

Синтаксис:

Велика теорема Ферма:

X <sup>n</sup> + Y <sup>n</sup> = Z <sup>n</sup>

де n – ціле число > 2

Результат:

Велика теорема Ферма:

X n + Y n = Z n

де n – ціле число > 2




Розмітка коду

<code> - призначений для відображення одного або декількох рядків тексту, який є програмним кодом.

Синтаксис:

<code> if (<ЛОГІЧНИЙ ВИРАЗ>) { ІНСТРУКЦІЯ 1 } else { ІНСТРУКЦІЯ 2 } </code>

Результат:

if (<логічний вираз>) { Інструкція 1 } else { Інструкція 2 }

<kbd> - використовується для позначення тексту, який набирається на клавіатурі або назви клавіш.

Синтаксис:

<kbd>ALT</kbd> + <kbd>F4</kbd> - це гарячі кнопки на команду закриття активного вікна програми.

Результат:

Alt + F4 - це гарячі кнопки на команду закриття активного вікна програми.

<samp> - використовується для відображення тексту, який є результатом виведення комп'ютерної програми чи скрипту.

Синтаксис:

Перевірка, чи підтримує браузер <samp>JAVASCRIPT 1.3 ЧИ НІ</samp>.

if (window.JS13) document.write("Ваш браузер підтримує JavaScript 1.3");

В результаті виконання скрипта ви побачите текст <samp>ВАШ БРАУЗЕР ПІДТРИМУЄ JAVASCRIPT 1.3</samp>, якщо браузер працює з версією 1.3.

Результат:

Перевірка, чи підтримує браузер JavaScript 1.3 чи ні.

if (window.JS13) document.write("Ваш браузер підтримує JavaScript 1.3");

В результаті виконання скрипта ви побачите текст Ваш браузер підтримує JavaScript 1.3, якщо браузер працює з версією 1.3.

<dfn> - цей елемент застосовується для виділення терміну при першій появі у тексті. Зазвичай текст відображається за допомогою курсивного накреслення.

Синтаксис:

<dfn>КАПІТЕЛЮ</dfn> в друкарні називається текст, набраний великими літерами зменшеного розміру.

Результат:

Капітелю в друкарні називається текст, набраний великими літерами зменшеного розміру.




Цитати, означення, аббревіатури

<abbr> - показує, що послідовність символів є абревіатурою. За допомогою цього атрибуту дається розшифровка скорочення, що дозволяє розуміти абревіатуру тим людям, які з нею не знайомі.

Синтаксис:

<abbr>CGI</abbr> позначається протокол, за допомогою якого будь-які зовнішні програми взаємодіють із веб-сервером. За допомогою <abbr>CGI</abbr> на сервері можна виконувати програми будь-якою мовою програмування і результат їхньої дії виводити у вигляді веб-сторінки.

Результат:

CGI позначається протокол, за допомогою якого будь-які зовнішні програми взаємодіють із веб-сервером. За допомогою CGI на сервері можна виконувати програми будь-якою мовою програмування і результат їхньої дії виводити у вигляді веб-сторінки.

<q> - використовується виділення у тексті цитат. Вміст контейнера автоматично відображається у браузері у лапках.

Синтаксис:

Ілон Маск говорив: <q>Люди сьогодні якось занадто захопилися всяким там інтернетом, фінансами і правом. Це одна із причин, чому у нас мало інновацій</q>.

Результат:

Ілон Маск говорив: Люди сьогодні якось занадто захопилися всяким там інтернетом, фінансами і правом. Це одна із причин, чому у нас мало інновацій.

<cite> - представляє назву творчої роботи (книги, статті, поеми, фільми, пісні та ін.). Це може бути твір, на який посилаються в цитаті, або просто робота, згадана побіжно.

Синтаксис:

Гра <cite>COUNTER-STIKE: GLOBAL OFFENSIVE</cite> - багатокористувацька гра, розроблена компаніями <cite>VALVE</cite> і <cite>HIDDEN PATH ENTERTAINMENT</cite> Випуск гри відбувся <cite>21 СЕРПНЯ 2012 РОКУ</cite>.

Результат:

Гра Counter-Strike: Global Offensive - багатокористувацька гра, розроблена компаніями Valve і Hidden Path Entertainment. Випуск гри відбувся 21 серпня 2012 року.

<blockquote> - призначений виділення довгих цитат всередині документа. Текст усередині відображається як вирівняний блок з відступами ліворуч та праворуч, а також з відбивкою зверху та знизу.

Синтаксис:

Джефф Безос розповів про три ідеї компанії Amazon:
<blockquote>
В основі Amazon лежать три ідеї, які дозволили компанії
домогтися успіху. Ставте клієнта на перше місце,
експериментуйте і будьте терплячими.
</blockquote>

Результат:

Джефф Безос розповів про три ідеї компанії Amazon:

В основі Amazon лежать три ідеї, які дозволили компанії
домогтися успіху. Ставте клієнта на перше місце,
експериментуйте і будьте терплячими.




Розриви

<br> - вказує місце перенесення рядка у тексті. Тег ставиться в кінці тексту, після чого буде виведено з нового рядка.

Синтаксис:

Вчіться, брати мої!<br> Думайте, читайте,<br> І чужому навчайтесь, -<br> Свого не цурайтесь:

Результат:

Вчіться, брати мої!
Думайте, читайте,
І чужому навчайтесь, -
Свого не цурайтесь:

<pre> - визначає блок попередньо форматованого тексту. Зазвичай такий текст відображається моноширинним шрифтом і з усіма пробілами між словами.

Синтаксис:

<pre>
Мені тринадцяте минало. Я пас ягнята за селом. Чи то так сонечко сіяло, Чи так мені чого було?
</pre>

Результат:

                Мені тринадцяте минало.
                Я пас ягнята за селом.
                Чи то так сонечко сіяло,
                Чи так мені чого було?

<hr> - малює горизонтальну лінію.

Синтаксис:

<hr>

Асимптота, звичайно, підтримує нормальний ряд, що розходиться, при цьому, замість 13 можна взяти будь-яку іншу константу.

</hr>

Результат:

Асимптота, звичайно, підтримує нормальний ряд, що розходиться, при цьому, замість 13 можна взяти будь-яку іншу константу.




Списки

<ul> - встановлює маркований список. Кожен пункт повинен починатися з <li>

Синтаксис:

<ul>
<li>Марс</li>
<li>Юпітер</li>
<li>Земля</li>
<li>Сонце</li>
<li>Космос</li>
</ul>

Результат:

  • Марс
  • Юпітер
  • Земля
  • Сонце
  • Космос

<ol> - встановлює нумерований список. Кожен елемент має починатися з <li>.

Синтаксис:

<ol>
<li>Понеділок</li>
<li>Вівторок</li>
<li>Середа</li>
<li>Четврег</li>
<li>П'ятниця</li>
</ol>

Результат:

  1. Понеділок
  2. Вівторок
  3. Середа
  4. Четврег
  5. П'ятниця

<li> - визначає окремий пункт списку. Зазвичай використовується для створення елементів пунктів

Синтаксис:

<ul>
<li>Гаррі Поттер</li>
<li>Герміона Ґрейнджер</li>
<li>Рональд Візлі</li>
</ul>

Результат:

  • Гаррі Поттер
  • Герміона Ґрейнджер
  • Рональд Візлі



Гіперпосилання

<a> - є одним з важливих HTML елементом і призначений для створення посилань. Для цього необхідно повідомити браузеру, що є посиланням, а також вказати адресу документа, на який слід зробити посилання.

Синтаксис:

<a href="ШЛЯХ"> НАЗВА</a>

Результат:

Посилання на сайт

Синтаксис:

<img src="ШЛЯХ" alt="НАЗВА">

Результат:




Спеціальні символи

&nbsp; – це спеціальний символьний код html розмітки, що відноситься до escape послідовностей. Спочатку йде амперсанд, потім пишеться ключове слово "nbsp" і в кінці крапка з комою.

Синтаксис:

слово&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;слово

Результат:

слово      слово

Мнемоніка (&copy;) - знак охорони авторського права, є латинською літерою C. Спочатку йде амперсанд, потім пишеться ключове слово "copy" і в кінці крапка з комою.

Синтаксис:

&copy;Контакти

Результат:

©Контакти

&trade; - знак торгової марки, є латинськими літерами T і M. Спочатку йде амперсанд, потім пишеться ключове слово "trade" і в кінці крапка з комою.

Синтаксис:

Почайна&trade;

Результат:

Почайна™

&amp; - символ амперсанда (&).

Синтаксис:

Клава Кока &amp; NILETTO - Краш

Результат:

Клава Кока & NILETTO - Краш

&lt; - Знак "менше ніж" (початок тега). Він використовуються для явного визначення символа менше.

Синтаксис:

2 &lt; 6

&lt;p>Молоко &lt;/p>

Результат:

2 < 6

<p>Молоко </p>

&gt; - знак "більше ніж" (кінець тега). Використовуються, щоб явно визначати більше символа.

Синтаксис:

10 &gt; 5

<p>Молоко<p>

Результат:

10 > 5

Молоко
