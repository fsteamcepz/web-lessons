основні властивості css для роботи з текстом

Властивість font-size

Властивість font-size змінює розмір шрифта.

Розмір шрифту задається за допомогою значення та одиниць виміру (em, px, pt, %)

Синтаксис:

CSS

p{
font-size: 20px;
}

HTML

<p>Hello, world!</p>

Результат:

Hello, world!

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту.

Може приймати наступні значення:

  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме, що і 700
  • lighter - змінює насиченість відносно бітьківського стилю в меншу сторону
  • bolder - змінює насиченість відносно бітьківського стилю в більшу сторону
  • 100,200,300,400,500,600,700,800,900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис:

CSS

.bold{
font-weight: bold;
}
.thin{
font-weight: 100;
}
.max{
font-weight: 900;
}

HTML

<p>
<span class = "bold">Напівжирний текст</span>
<span class = "thin">найтонший текст</span>
<span class = "max">найтовщий текст</span>
</p>

Результат:

Напівжирний текст, найтонший текст, найтовщий текст.

Властивість font-style

Властивість font-style визначає, яким чином шрифт повинен бути стилізований, чи це normal, italic, або oblique.

Може приймати наступні значення:

  • normal - виділяє шрифт, який класифікований як нормальний
  • italic - виділяє шрифт курсивом. Якщо не доступна курсивна версія шрифту, для заміни використовується oblique класифікація
  • oblique - виділяє шрифт косим. Якщо не доступна коса версія шрифту, для заміни використовується italic класифікація

Синтаксис:

CSS

.normal {
font-style: normal;
}
.italic {
font-style: italic;
}
.oblique {
font-style: oblique;
}

HTML

<span class = "normal">Цей абзац нормальний</span>
<span class = "italic">Цей абзац виділено курсивом</span>
<span class = "oblique">Цей абзац є косим</span>

Результат:

Цей абзац нормальний.

Цей абзац виділено курсивом.

Цей абзац є косим.

Властивість font-family

Властивість font-family визначає список шрифтів у пріоритеті від найвищого до нижчого.

Може приймати наступні значення:

  • serif - гліфи мають завершальні штрихи, що розширюються або звужуються кінці
  • sans-serif - гліфи мають гладкі закінчення
  • monospace - усі гліфи мають однакову фіксовану ширину

Синтаксис:

CSS

.serif {
font-family: Times, Times New Roman, Georgia, serif;
}
.sansserif {
font-family: Verdana, Arial, Helvetica, sans-serif;
}
.monospace {
font-family: Lucida Console, Courier, monospace;
}

HTML

<span class = "serif">Це приклад шрифту serif</span>
<span class = "sansserif">Це приклад шрифту sans-serif</span>
<span class = "monospace">Це приклад шрифту monospace</span>

Результат:

Це приклад шрифту serif

Це приклад шрифту sans-serif

Це приклад шрифту monospace

Властивість color

Властивість color змінює колір шрифта.

Ця властивість може приймати як однорідний колір так і прозорість, але на відміну від зображень не може бути градієнтом.

Може приймати наступні значення:

  • color - колір тексту
  • initial - встановлює властивість у значення без задання
  • inherit - вказує на спадковість властивості від свого батьківського елемента
  • transparent - робить текст прозорим (невидимим)

Синтаксис:

CSS

.name{
color: green;
}
.hex{
color: #F12668;
} .rgb{
color: rgb(72, 102, 157);
}
.rgba{
color: rgba(72, 102, 157, 0.45);
}
.hsl{
color: hsl(0, 100%, 30%);
}
.hsla{
color: hsla(0, 60%, 77%, 0.7);
}

HTML

<p class="name">green</p>
<p class="hex">#F12668</p>
<p class="rgb">rgb(72, 102, 157)</p>
<p class="rgba">rgba(72, 102, 157, 0.45)</p>
<p class="hsl">hsl(0, 100%, 30%)</p>
<p class="hsla">hsla(0, 60%, 77%, 0.7)</p>

Результат:

green

#F12668

rgb(72, 102, 157)

rgba(72, 102, 157, 0.45)

hsl(0, 100%, 30%)

hsla(0, 60%, 77%, 0.7)

Властивість text-transform

Властивість text-transform визначає, якими літерами буде відображено текст - ВЕЛИКИМИ, малими або Кожне Слово З Великої Літери..

Може приймати наступні значення:

  • none - регіст символів не буде змінено
  • capitalize - перший символ кожного слова в реченні буде заголовним
  • uppercase - всі символи тексту стають великими (верхній регістр)
  • lowercase - всі символи тексту стають малими (нижній регістр)

Синтаксис:

CSS

.up{
text-transform: uppercase;
}
.low{
text-transform: lowercase;
}
.cap{
text-transform: capitalize;
}

HTML

<p class="up">Я просто текст, нічого незвичного</p>
<p class="low">Я просто текст, нічого незвичного</p>
<p class="cap">Я просто текст, нічого незвичного</p>

Результат:

Я просто текст, нічого незвичного

Я просто текст, нічого незвичного

Я просто текст, нічого незвичного

Властивість text-decoration

Властивість text-decoration визначає оздоблення тексту.

Може приймати наступні значення:

  • none - звичайний текст
  • underline - підкреслений текст
  • overline - лінія над текстом
  • line-through - закреслений текст
  • blink - блимання тексту (не підтримується всіма браузерами)

Синтаксис:

CSS

.underline{
text-decoration: underline red;
}
.overline{
text-decoration: wavy overline lime;
}
.through{
text-decoration: line-through;
}
.none{
text-decoration: none;
}
.blink{
text-decoration: blink;
}

HTML

<p class="underline">Текст підкреслений синім кольором</p>
<p class="overline">Текст, а над ним лінія</p>
<p class="through">Перекреслений текст</p>
<p>Це <a class="none" href="#">посилання не підкреслено</a>, а має бути за замовчуванням.</p>
<p class="blink">Цей текст блимає. Але це залежить від того, чи підртримує цю властивість твій браузер...</p>

Результат:

Текст підкреслений синім кольором

Текст, а над ним лінія

Перекреслений текст

Це посилання не підкреслено, а має бути за замовчуванням.

Цей текст блимає. Але це залежить від того, чи підртримує цю властивість твій браузер...
